# ulises-c.github.io

Personal portfolio site for Ulises Chavarria — hosted on GitHub Pages at [ulises-c.github.io](https://ulises-c.github.io/).

## Stack

- Vanilla HTML, CSS, JavaScript — no build step, no frameworks
- Hosted via GitHub Pages (root of `master` branch)
- Google Fonts: Inter + JetBrains Mono

## Structure

```
.
├── index.html       # Single-page portfolio
├── style.css        # All styles (dark theme, responsive)
├── script.js        # Typewriter, scroll animations, mobile nav
└── assets/
    ├── resume.pdf   # Resume (add before going live)
    └── headshot.*   # Photo (add before going live)
```
